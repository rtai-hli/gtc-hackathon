Demo 2: shape-outside & ::first-letter

This text demonstrates the '::first-letter' pseudo-element, creating a stylish drop cap. More importantly, notice how the text gracefully wraps around the circular element on the left. This is achieved using 'shape-outside: circle()'. It allows content to flow along the contour of complex shapes, not just boring rectangles. This is a powerful tool for creating dynamic, magazine-style layouts directly in CSS without complex hacks. As you resize the window, you'll see the text reflow perfectly around the circle's defined shape, maintaining a clean and professional margin.
